## 概要

好きなポケモンをいつでもARで見れるサイトです。

PokeAPIというポケモン画像を取得できるAPIを利用しています。

個人的にポケモンが好きなのと、娘に喜んでもらうために作成しました。

## 特徴

- 手軽にAR体験
- ポケモン1025種類（2024.1.17時点）
- 色違いポケモンあり

## 使用技術

- Ruby 3.3.0
- Ruby on Rails 7.1.2
- Rspec
- MySQL 8
- Docker
- AWS(ECS, RDS, ALB)
- TailwindCSS
